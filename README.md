# Yorkshire Bus Network Analysis

## Project Overview
This project analyzes Yorkshire GTFS bus network data using SQL and Tableau.

## Tools Used
- MySQL
- Tableau Public
- GTFS Dataset
- GitHub

## Key Findings

### Top Routes by Total Trips
- YELL recorded the highest number of trips.
- BLUE was the second busiest route.

### Busiest Stops
- Bradford Interchange Bus Station received the highest stop visits.
- Huddersfield Bus Station and Leeds City Bus Station also showed high activity.

### Peak Hour Analysis
- Bus activity increases rapidly after 6 AM.
- Network activity peaks around midday.
- Service activity gradually decreases after evening hours.

### Route Coverage
- Station Road had the highest route connectivity.
- Route 7 served the largest number of stops across the network.
